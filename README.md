SETUP:

1) install ffmpeg=python using the following command:
        pip install ffmpeg-python
2) move ffmpeg.exe to base of workspace
3) in ot_converter.py, change input_directory and output_directory variables to directories of your choosing.


Sometimes program will have an error at the paths of input/output directories. If so, insert double backslashes 
instead of single backslashes and your problem should resolve.
